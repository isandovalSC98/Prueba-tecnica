Pasos a seguir para la instalacion del proyecto.

1- Tener PosgreSQL en su version 18.
2- Contar con PHP Version 8.0.30
3- Instalar Xampp en la maquina
4- Una vez instalado Xampp, bajar el repositorio del proyecto PHP.
5- Dejar el proyecto en la ruta de instalacion de Xampp, por lo general disco local C, en mi caso es C:\xampp\htdocs\PruebaTecnicaPHP
6-Crear una base de datos en PosgreSQL 18 llamada "Desis".
7-Bajar la carpeta "Base de datos" de github el cual contendra la base de datos para el proyecto.
8-Una vez creada la base de datos, importar el backup del archivo descargado previamente para cargar la base creada y ejecutar.
9-Ejecutar en Xampp la opcion de apache y en el navegador escribir lo siguiente "http://localhost/PruebaTecnicaPHP/".
10- Si todo esta correcto deberia de ejecutar el proyecto de forma correcta.
